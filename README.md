# List of Applications

# 1. Explorer Browser

- HTML file: `app1/Browser.html`  
- Description:  
  This application simulates the Windows 7 Explorer browser. The entire interface is built with HTML and CSS to recreate the classic Windows 7 style.

# 2. Calculator

- HTML file: `app2/calculadora.html`  
- Description:  
  This application simulates a desktop calculator. It provides functionality for addition, subtraction, multiplication, and division. Some buttons are purely decorative (mocked).  
  The background uses a Windows-like style to give the feeling of opening a new system window.  
  It is built with HTML, CSS, and a JavaScript file responsible for the calculation logic.  
  The calculator logic is based on the following YouTube tutorial:
