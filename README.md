# Descripcion
En este mini sistema operativo quisimos replicar las aplicaciones de la galeria, un block de notas, un reproductor de videos, un navegador y una calculadora, en lasreplicas puedes agregar texto, pero la informacion de lo que escribas no se va a guardar, la mas funcional es la calculadora, esta suma, resta, multiplica y divide, algunos botones son maquetados, en el reproductor de videos contamos con un trailer, en la galeria contamos con varias imagenes y en el block de notas puedes escribir texto, pero no lo puedes guardar.
